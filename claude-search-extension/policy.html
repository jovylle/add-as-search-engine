Privacy Policy

Effective Date: October 14, 2025

TL;DR: This extension is completely privacy-focused. It doesn't collect, store, or transmit any personal data. It simply adds Claude AI as a search option in Chrome.

1. Information We Collect

None. This extension does not collect, store, or transmit any personal information, browsing data, search queries, or any other user data.

2. How the Extension Works

This extension only:

  • Adds Claude AI as a search engine option in Chrome
  • Enables the "claude" keyword for quick access
  • Redirects your search queries directly to Claude AI's website

The extension operates entirely locally within your browser and does not communicate with any external servers except Claude AI when you choose to search.

3. Data Sharing

Since we don't collect any data, there is nothing to share. When you search using this extension, your query goes directly to Claude AI's servers, subject to Anthropic's privacy policy.

4. Permissions

This extension only requests the "search" permission, which is necessary to add Claude AI as a search engine option. No other permissions are requested or used.

5. Third-Party Services

When you use this extension to search, you'll be interacting with:

  • Claude AI (Anthropic): Subject to their Privacy Policy

6. Changes to This Policy

If we make changes to this privacy policy, we will update the effective date above. Since the extension doesn't collect data, significant changes are unlikely.

7. Contact Information

If you have questions about this privacy policy, please visit jovylle.com for contact information.

8. Open Source

This extension is open source. You can review the complete source code on GitHub to verify our privacy claims.
